Run performance tests\n   npm run test:performance -- --base-url=https://blue.yourdomain.com\n   ```\n\n3. **Switch Traffic**\n   ```bash\n   # Update load balancer or DNS\n   # Switch Cloudflare DNS to point to blue environment\n   ```\n\n4. **Monitor & Rollback if Needed**\n   ```bash\n   # Monitor for 10 minutes\n   # If issues detected, switch back to green\n   ```\n\n### Automated Blue-Green (GitHub Actions)\nThe CI/CD pipeline automatically handles blue-green deployments:\n- Builds new version in \"blue\" environment\n- Runs health checks and performance tests\n- Switches traffic if all checks pass\n- Monitors for issues and auto-rollback if needed\n\n## ðŸ†˜ Rollback Procedures\n\n### Quick Rollback (DNS)\n```bash\n# 1.

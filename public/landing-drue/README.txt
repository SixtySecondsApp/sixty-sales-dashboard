Sixty Seconds Landing Page - Complete Package
==============================================

This package contains everything needed to view the landing page.

QUICK START:
------------
1. Extract all files from this zip
2. Open a terminal in this folder
3. Run: node serve-dist.js
4. Open http://localhost:3000 in your browser

ALTERNATIVE METHODS:
--------------------
- Python 3: python3 -m http.server 3000
- Python 2: python -m SimpleHTTPServer 3000
- npx: npx serve .

FILES INCLUDED:
---------------
- index.html          Main landing page file
- assets/             JavaScript and CSS bundles
- favicon_*.png       Favicon files
- manifest.json       PWA manifest
- serve-dist.js       Simple HTTP server script
- EXPORT_INSTRUCTIONS.md  Detailed instructions

IMPORTANT:
----------
This is a React Single Page Application (SPA) that requires
a web server to run. You cannot simply open index.html directly
in a browser - it must be served via HTTP.

For questions or issues, refer to EXPORT_INSTRUCTIONS.md
